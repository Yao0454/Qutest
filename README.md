# 👾 蛆度测试

> 你到底有多蛆？经过严肃的伪科学研究，我们终于能给你一个答案。

---

## 这是什么

一个测试"你有多蛆"的网页小测试。

从四个维度对你进行多维度蛆性分析，外加一个隐藏的自卑系数静默加权——同样的行为，自卑驱动的蛆比单纯懒惰的蛆，蛆得更彻底。

共 **30 题**，随机抽取，每次测试题目不完全相同。答完给出你的蛆属等级认定书，从「清醒的人类 🌿」到「宇宙级蛆神降临 👾」，共 8 级，覆盖人类堕落的完整光谱。

## 测试维度

| | 维度 | 测什么 |
|--|------|--------|
| 🛋️ | **身体荒废度** | 运动频率、久坐时长、洗澡周期、健身卡吃灰、外卖取餐意愿、房间考古价值... |
| 📱 | **数字沉溺度** | 短视频日均时长、收藏坟场规模、睡前刷屏到几点、被算法驯化程度、焦虑性刷屏... |
| 🏚️ | **社会脱轨度** | 待办积压年限、简历最后更新时间、励志视频感动后的行动率、"等状态好了"说了多久... |
| 🫥 | **社交萎缩度** | 已读不回率、外卖备注字数、群聊幽灵程度、陌生来电处理方式、节日祝福零回复... |
| 🔍 | **自卑指数** | 隐藏维度，不单独展示，作为权重系数乘到以上四个维度上。自卑越深，蛆力值越高。 |

---

## 评分机制

```
各维度加权分 = 原始维度分 × 自卑乘数（1.0 ~ 1.4）
最终蛆力值   = 四个加权维度分之和（满分 125）
```

自卑题随机穿插在 30 题中，题目分类显示为「日常随感」，用户无法察觉。

---

## 题库规模

| 类型 | 题库总量 | 每次抽取 |
|------|---------|---------|
| ABCD 行为题 | 100 题（每维度 25 题） | 25 题 |
| E 自卑题 | 20 题 | 5 题 |
| **合计** | **120 题** | **30 题** |

每题 5 个选项，分值为 `0 / 1 / 2 / 4 / 5`（跳过 3，拉开高分区分度）。

---

## 抗乱选机制

- **速度检测**：答题过快触发惩罚系数
- **模式识别**：全选同一选项会被识破并修正评分
- **边缘检测**：全选第一或最后一项触发减分
- **实时提示**：检测到异常行为立即弹出警告
