## 这是什么？

一个测试"你有多蛆"的网页小测试。

从 **四个维度** 对你进行多维度蛆性分析，随机抽取25题，最终给出你的蛆属等级认定书——从"清醒的灵魂🌸"到"宇宙级蛆神👾"，共9个等级，覆盖人类堕落的全部光谱。

## 特性

- **100题题库，每次随机抽25题**，重复测试也不会审美疲劳
- **抗乱选机制**：速度检测 + 模式识别，乱点一通会被系统识破并惩罚评分
- **9级蛆属认定**，每级都有专属的风趣嘲讽文案
- **四维度独立评分**，测完可以看自己在哪个方向蛆得最彻底
- **一键复制分享**，方便传播让朋友也来确认自己的蛆性
- **纯静态单文件**，无后端、无数据库、无隐私收集，下载即用

## 部署

整个项目就是一个 `index.html`，扔到任何地方都能跑。

**推荐：Cloudflare Pages（免费）**

1. 把文件改名为 `index.html`
2. 上传到 [Cloudflare Pages](https://pages.cloudflare.com)
3. 得到一个 `xxx.pages.dev` 链接，完事

其他平台同理：GitHub Pages、Vercel、Netlify 均可，原理一样。

有自己服务器的话：

```bash
scp index.html root@你的IP:/var/www/html/index.html
```
